基本电学原理

Fundamental Electrical Principles - 电子学基础理论

欧姆定律

欧姆定律是电学中最基本的定律，描述了电压、电流和电阻之间的关系。

V = I × R

其中：
V = 电压 (伏特, V)
I = 电流 (安培, A)
R = 电阻 (欧姆, Ω)

欧姆定律的变形形式：

  • I = V / R - 电流等于电压除以电阻
  • R = V / I - 电阻等于电压除以电流
适用条件：欧姆定律适用于线性电阻，即电阻值不随电压或电流变化的元件。

基尔霍夫定律

基尔霍夫电流定律(KCL)

在任意时刻，流入节点的电流之和等于流出节点的电流之和。

∑I_in = ∑I_out

例如：I1 + I2 = I3 + I4

基尔霍夫电压定律(KVL)

在任意闭合回路中，电压降之和等于电压升之和。

∑V_drop = ∑V_rise

例如：V1 + V2 + V3 = V_source

功率和能量

电功率是单位时间内所做的电功。

P = V × I = I² × R = V² / R

其中：
P = 功率 (瓦特, W)
V = 电压 (伏特, V)
I = 电流 (安培, A)
R = 电阻 (欧姆, Ω)

电能计算：

W = P × t = V × I × t

其中：
W = 电能 (焦耳, J)
P = 功率 (瓦特, W)
t = 时间 (秒, s)

常用单位换算：
1 kWh = 3.6 × 10^6 J

电阻串并联

串联电阻

R_total = R1 + R2 + R3 + ... + Rn
  • 总电阻等于各电阻之和
  • 电流处处相等
  • 电压按电阻值分配

并联电阻

1/R_total = 1/R1 + 1/R2 + 1/R3 + ... + 1/Rn

或等效为：
R_total = R1 × R2 / (R1 + R2)  (仅适用于两个电阻)
  • 总电阻小于最小的单个电阻
  • 电压处处相等
  • 电流按电阻值分配

电容特性

电容器储存电荷的能力用电容值表示。

C = Q / V

其中：
C = 电容 (法拉, F)
Q = 电荷量 (库仑, C)
V = 电压 (伏特, V)

常用单位：
1 F = 10^6 μF = 10^9 nF = 10^12 pF
  • 容抗：Xc = 1 / (2πfC) - 电容对交流电的阻碍作用
  • 储能：W = ½CV² - 电容器储存的能量
  • 隔直流通交流 - 电容允许交流通过，阻断直流

电感特性

电感器储存磁场能量的能力用电感量表示。

V = L × (dI/dt)

其中：
L = 电感 (亨利, H)
V = 电压 (伏特, V)
dI/dt = 电流变化率 (A/s)

常用单位：
1 H = 10^3 mH = 10^6 μH
  • 感抗：Xl = 2πfL - 电感对交流电的阻碍作用
  • 储能：W = ½LI² - 电感储存的能量
  • 阻交流通直流 - 电感阻碍交流通过，允许直流通过

RC和RL时间常数

RC时间常数

τ = R × C

其中：
τ = 时间常数 (秒, s)
R = 电阻 (欧姆, Ω)
C = 电容 (法拉, F)

充电/放电过程：
- 5τ后达到最终值的99.3%
- 3τ后达到最终值的95%

RL时间常数

τ = L / R

其中：
τ = 时间常数 (秒, s)
L = 电感 (亨利, H)
R = 电阻 (欧姆, Ω)
